Ejecución: ant run

	En seguida se solicita un entero 'k' entre 1 y 20.
	
	El programa genera un arreglo de tamaño 50k. Dicho arreglo se guarda en
	un archivo llamado 'arreglo.txt'.
	
	Después se genera, pseudoaleatoriamente, una clave para buscar dentro
	de dicho arreglo.
	
	La clave pertenece al útimo intervalo de búsqueda del algoritmo de
	búsqueda exponencial, el cual representa el peor de los casos de la
	complejidad de dicho algoritmo.
	
	El programa finaliza indicando si la clave pertenece o no al
	arreglo y muestra el índice del arreglo en donde se úbica la clave.
