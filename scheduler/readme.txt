Ejecución:

	Utilizando ant, la ejecución sería: ant run


	En seguida se solicita, una a una, las tareas a evaluar.
	La manera de ingresarlas es mediante dos enteros separados
	por una sola comma, ejemplo:

	Tarea 1:   1, 3
	Tarea 2:   2, 4
	Tarea 3:   2, 5

	Para terminar de agregar tareas, dar ENTER con la cadena vacía.
